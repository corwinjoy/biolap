========================================================================
      SEQ Schema
========================================================================

This is a very simple demo schema containing three tables, 
ORGANISMS, PROTEINS and SEQUENCES.  The data is held in the SEQUENCES table
a consists of 15 rhodopsin like GPCR sequences with taxonomy information
in the ORGANISMS table and functional information in the PROTEINS table.

To set the schema up run

	schema.sql
	data.sql
